Principais formatações

Negrito / Destaque

Nessa frase temos uma palavra em Negrito usando a tag <b> (não semantica, pois é somente estilo)

Para usar de forma semantica se usa a tag <strong> para atribuir sentido (mostrar que é uma frase forte e de destaque com algum significado alem de estilo)

Itálico / Enfase

Nesta frase, temos uma palavra em Itálico usando a tag <i> (não semantica)

Nesta frase temos uma palavra em enfâse usando a tag <em> (semantica)

Texto marcado

Podemos criar um texto marcado com a tag MARK

E no outro paragrafo, temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado (tag del)

Texto inserido

Pode marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois (tag ins).
